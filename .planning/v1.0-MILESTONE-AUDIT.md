## Cross-Phase Integration

28/33 integration paths fully wired. 5 paths have issues (no blockers, all degraded gracefully):

### High Priority (broken production behavior)

**1. body_html never populated**
- **Flow:** Gmail → GmailPoller → Pipeline → Email.body_html
- **Issue:** `EmailMessage` DTO has no `body_html` field. GmailPoller doesn't extract HTML body. Pipeline doesn't save it. HTML emails render as plain text in the detail panel.
- **Impact:** DASH-03 degraded — works but HTML formatting lost
- **Files:** dtos.py, gmail_poller.py, pipeline.py

**2. chat_webhook_url not in SystemConfig seed**
- **Flow:** assign_email() → ChatNotifier → Chat webhook
- **Issue:** Web container's assignment service reads `chat_webhook_url` from SystemConfig. Key not seeded. No env var fallback for web container.
- **Impact:** ASGN-05 degraded — assignment Chat notifications silently skip unless manually configured
- **Files:** assignment.py, 0002_seed_default_config.py

### Medium Priority (broken UX, not a crash)

**3. Quiet hours config type mismatch**
- **Issue:** Seeded as `int` (20, 8) but ChatNotifier parses `"%H:%M"` format. Falls through to safe default (quiet hours inactive). All notifications fire at all hours.
- **Impact:** SLA-04 degraded — breach alerts always go through regardless of time
- **Fix:** Change seed to `str` type with `"20:00"`/`"08:00"` values, or update ChatNotifier to handle int

**4. accept_ai_suggestion missing OOB card update**
- **Issue:** Returns only detail panel render, no OOB card HTML. AI badge persists on card until page refresh.
- **Impact:** ASGN-04 display only — assignment works correctly, just visual stale badge

### Low Priority (dead code / cosmetic)

**5. Dead DashboardView** — accounts/DashboardView never served (root RedirectView intercepts)
**6. Activity timeline colors** — Phase 4 actions (auto_assigned, claimed, sla_breached, priority_bumped) all render as slate dots
