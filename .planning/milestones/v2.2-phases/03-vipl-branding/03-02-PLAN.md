class ChatNotifier:
    def __init__(self, webhook_url: str)
    def _post(self, payload: dict) -> bool
    def _is_quiet_hours(self) -> bool
    def notify_assignment(self, email, assignee) -> bool      # Card: assignment alert
    def notify_new_emails(self, emails) -> bool                # Card: poll summary
    def notify_breach_summary(self, summary_data: dict) -> bool # Card: SLA breach manager summary
    def notify_eod_summary(self, stats: dict) -> bool          # Card: daily EOD summary
    def notify_personal_breach(self, assignee_name: str, breached_emails: list) -> bool  # Card: personal breach alert

Each method builds a `card` dict with `header` and `sections`, then wraps in `cardsV2` payload.

Google Chat Cards v2 header format:
  "header": {
      "title": "...",
      "subtitle": "...",
      "imageUrl": "https://...",      # ADD THIS
      "imageType": "CIRCLE",           # ADD THIS
      "imageAltText": "VIPL Logo"      # ADD THIS
  }

Footer: `fixedFooter` only works in dialogs, NOT webhook messages.
Use a textParagraph widget in the last section instead:
  {"textParagraph": {"text": "<i>Sent by VIPL Email Triage</i>"}}

The imageUrl MUST be the production static URL:
  https://triage.vidarbhainfotech.com/static/img/vipl-icon.jpg

Use `tracker_url` from SystemConfig to build this dynamically:
  tracker_url = SystemConfig.get("tracker_url", "https://triage.vidarbhainfotech.com")
  icon_url = f"{tracker_url}/static/img/vipl-icon.jpg"
