# Code of Conduct

## Our Standards

This is an internal project of Vidarbha Infotech Private Limited. All contributors are expected to:

- Be respectful and professional in all communications
- Focus on constructive feedback and collaboration
- Follow the project's coding standards and review processes
- Report issues through proper channels

## Scope

This applies to all project spaces: code, issues, pull requests, and related communications.

## Enforcement

Violations should be reported to shreyas@vidarbhainfotech.com. All reports will be reviewed promptly.
